# uog navigetor (Flutter frontend)

This folder contains a minimal Flutter scaffold for the `uog navigetor` app.

Quick start (Windows):

1. Install Flutter SDK: https://flutter.dev/docs/get-started/install
2. From this folder, run:

```powershell
flutter pub get
flutter run -d windows
```

Notes:
- The sample `lib/main.dart` provides a simple list of locations and opens Google Maps via `url_launcher`.
- Add more dependencies (maps, geolocation) as needed in `pubspec.yaml`.
